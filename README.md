# Alisveris-Sepeti-Arayuz-Tasarimi
Java Swing kullanarak geliştirdiğim bu proje, bir alışveriş sepetinin temel işleyişini ve görsel yapısını sunar. Kullanıcıların ürünleri listelemesi ve seçmesi üzerine odaklanan bu çalışma, yazılım mühendisliği eğitimimdeki arayüz geliştirme ve nesne yönelimli programlama pratiklerimin bir parçasıdır. Anlaşılır ve sade bir tasarıma sahiptir.
